# 关于本书

本书是基于[《the Book of Shaders》](https://thebookofshaders.com/?lan=ch)的扩展，基于原作者[Patricio Gonzalez Vivo](http://patriciogonzalezvivo.com/)创作内容的基础上进行二次创作，相关内容遵循原作者开源协议和MIT协议。


这是一本关于 Fragment Shaders（片段着色器）的入门指南，它将一步一步地带你领略其中的纷繁与抽象。

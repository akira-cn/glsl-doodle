## 内建 uniforms

除了内建模块，glsl-doodle还提供了一组以dd_前缀开头的内建uniforms，我们之前已经见过的dd_resolution和dd_time就属于内建uniforms。

让我们来详细看一下：

---

| uniform | 说明 |
| --- | --- |
| uniform vec2 dd_resolution | 画布分辨率，比如画布大小200x200，那么dd_resolution的值就是vec2(200, 200) |
| uniform float dd_time | 渲染每一帧时的时间戳，方便用来实现动画 |
| uniform fload dd_timeDelta | 渲染当前帧距离上一帧经过的时间 |
| uniform float dd_frameIndex | 当前渲染帧的序号，从1开始，每渲染一次增加1 |
| uniform float dd_randseed | 随机种子，每次渲染都会随机生成一个，第一帧渲染时dd_randseed等于dd_randseed0 |
| uniform float dd_randseed0 | 随机种子，第一帧渲染时生成 |
| uniform vec4 dd_mousePosition | 鼠标移动时，xy传入鼠标的归一化坐标，zw传入buttons按键，z是mousedown按键，w是click按键，否则值为vec4(vec2(-1), vec2(0)) |
| uniform vec3 dd_mouseDown | 保存最后一次按下鼠标的坐标和按键 |
| uniform int dd_mouseRec | 与[shadertoy](https://www.shadertoy.com/view/Mss3zH)的iMouse兼容的mouse事件 |
| uniform int dd_keyEvent | 键盘按下后，dd_keyCode.x为实际键码，dd_keyCode.y的值为1，键盘松开时，dd_keyCode.y值为2，其余情况值为0 |
| unifrom int dd_touchEvent | touch事件，xy入鼠标的归一化坐标，z当touchstart时，值为1，touchend时，值为3，touchmove时，值为2，其余情况该值为0 |

通过这些内建uniforms，我们可以很容易地实现动画、处理事件等等。
